This is a simulation of Conway's Game of Life. You can customize the size of the grid by adjusting the height and width (5x5 up to 25x25). The Play button will run the simulation, Step will run one turn, and New will create a new blank grid according to your specifications. Clicking on a turn number on the right will revert the grid back to that turn. Enjoy!
